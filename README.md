# Animation Technique Demos — Five Reels

Five ~30-second animation demos, each built with a different technique, themed around
the course **Human Wisdom for the Age of AI**. Built by Claude for Mustafa Akben.

**Live gallery:** https://mustafaakben.github.io/animation-demos/

| Reel | Technique | Piece |
|------|-----------|-------|
| 01 | HTML + CSS keyframes + JS | *The Delivery* — packages reveal the course's four capacities |
| 02 | p5.js generative art (seeded) | *Deliberate Currents* — interactive flow field, swarm vs. sages |
| 03 | Pagecast browser recording → ffmpeg | Reel 01 as MP4 + GIF |
| 04 | Python/PIL → Slack-validated GIF | *The Unboxing* — 480px message GIF + 128px emoji |
| 05 | Browser automation + screen capture | *The Packing Station* — an agent drives a live form |

`02-philosophy.md` holds the algorithmic philosophy behind Reel 02.
`tools/make_slack_gif.py` renders Reel 04 frame by frame.
