INSPECT opens the parcel on your desk. Read the M-6 draft; open the SOURCE FILE to verify it. Then stamp: APPROVE ships it, FLAG sends it to human escalation.

There is no timer — only the belt. It holds six parcels. When a seventh arrives, the oldest waiting parcel auto-clears as approved. Not deciding is a decision.

Fast, unverified approvals build a throughput multiplier. Whether throughput is the thing worth maximizing is, in a sense, the entire question.

Red mail is a consequence. Read it.
