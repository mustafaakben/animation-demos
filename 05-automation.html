An Agent Works the Station

No human touched this form. An automation script drove a real browser — clicking the field, typing character by character, picking from the dropdown, pressing the button — while the screen was recorded. Four labels, four shipments, 4/4 on the counter.

// the actual action script, abridged — one round of four
{ click: "#ship-to" }
{ type: "Class of 2026", delay: 70ms/char }
{ select: "#capacity" → "Judgment" }
{ click: "#pack" }
{ wait: 4300ms } // parcel drops, labels itself, ships left
